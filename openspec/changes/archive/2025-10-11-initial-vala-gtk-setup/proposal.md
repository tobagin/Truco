## Why
To set up the initial project structure for a GNOME GTK4/LibAdwaita application called Truco, written in Vala, using Blueprint for UI, Meson for building, and Flatpak for packaging.

## What Changes
- Add project structure with necessary directories and files for a Vala GTK application.
- Configure Meson build system.
- Set up Flatpak packaging for Flathub distribution.
- Define UI using Blueprint files.

## Impact
- Affected specs: project-structure, build-system, ui-framework, packaging.
- Affected code: New files and directories for the application.
